# Atenea — Organigrama (Web) (Mobile-first)

Este ZIP incluye **Dirección Política** y **Dirección Ejecutiva** con el mismo diseño “como antes”,
pero ahora **mobile-first** (sin scroll horizontal en móvil).

## Abrir en local
1) Descomprime el ZIP
2) Abre `index.html` con Chrome/Edge/Firefox

## Notas
- En móvil: los bloques se apilan en una columna y los conectores se ocultan para legibilidad.
- En desktop (>=900px): vuelve el layout original en 2 columnas con conectores y ancho de nodos consistente.
